SMS E-mail Voz

Praticidade para os desenvolvedores. Veja um exemplo:

curl -X POST \ https://api-messaging.movile.com/v1/send-sms \ -H 'authenticationtoken: <authenticationtoken>' \ -H 'username: <username>' \ -H 'content-type': "application/json' \ -d '{"destination": "5511984541027" , "messageText": "linha\nquebrada"}'

Como resposta, você receberá um JSON informando o ID que foi gerado para essa mensagem. (Resposta ou resposta síncrona da Movile):

[ { "id":"9cb87d36-79af-11e5-89f3-1b0591cdf807" } ]
